# 🧮 수치해석 (Numerical Analysis) 오리엔테이션

이 저장소는 **수치해석** 교과목의 실습 코드와 학습 자료를 관리하는 공간입니다. 
컴퓨터를 이용해 수학적 문제를 해결하는 알고리즘을 구현하고 실습합니다.

## 📅 교과목 개요
- **강의명**: 수치해석 (Numerical Analysis)
- **도구**: Python (NumPy, SciPy, Matplotlib) 또는 MATLAB
- **목표**: 오차 분석, 선형 방정식 해법, 보간법, 수치 미적분 등 핵심 알고리즘 이해

## 📂 저장소 구조
- `/Lecture`: 강의 슬라이드 및 요약 노트
- `/Assignments`: 과제 제출 폴더
- `/Practice`: 수업 시간 실습 코드 (`.py` 또는 `.ipynb`)
- `/Projects`: 기말 팀 프로젝트

## 📝 학습 내용
1. **오차론 (Error Analysis)**: 반올림 오차 및 절단 오차
2. **비선형 방정식의 해**: 이분법(Bisection), 뉴턴법(Newton-Raphson)
3. **선형 연립 방정식**: 가우스 소거법, LU 분해
4. **보간법 및 근사**: 라그랑주 보간법, 최소제곱법
5. **수치 미분과 적분**: 사다리꼴 공식, 심슨 공식
6. **상미분 방정식**: 오일러 방법, 룬게-쿠타(Runge-Kutta)
